# Lab: 화씨온도 변환기



## 1. 실습내용 



* input(), print()를 이용하여 화씨 온도 변환 프로그램을 만들어 보자

```text
# 변환 공식

화씨온도 = (섭씨온도 * 1.8) +32
```



## 2. 실행결과

```text
본 프로그램은 섭씨온도를 화씨온도로 변환하는 프로그램입니다.
변환하고 싶은 섭씨온도를 입력하세요.
32.2
섭씨온도: 32.2
화씨온도: 89.96
```



## 3. 문제해결

```python
print('본 프로그램은 섭씨온도를 화씨온도로 변환하는 프로그램입니다.')
print('변환하고 싶은 섭씨온도를 입력하세요.')

celsius = input()
fahrenheit = (float(celsius) * 1.8) + 32

print('섭씨온도:', celsius)
print('화씨온도:', fahrenheit)
```
